Note: This demo uses the production Octothorpes API at https://octothorp.es. Results will vary based on what's currently indexed in the protocol.

Basic Examples

1. Pages tagged with terms (auto-load)
Get pages tagged with "demo" - loads automatically on page load 
<octo-query 
  what="pages" 
  by="thorped" 
  o="demo"
  limit="5"
  autoload="true">
</octo-query>
2. Manual load button
Same query but requires clicking the load button (autoload="false") 
<octo-query 
  what="pages" 
  by="thorped" 
  o="indieweb"
  limit="5">
</octo-query>
3. Grid layout
Display results in a grid with multiple terms 
<octo-query 
  what="pages" 
  by="thorped" 
  o="web,protocol"
  layout="grid"
  limit="6"
  autoload="true">
</octo-query>
4. Compact layout without metadata
Minimal display - just titles and links 
<octo-query 
  what="pages" 
  by="posted" 
  limit="8"
  layout="compact"
  showmeta="false"
  autoload="true">
</octo-query>

Advanced Queries

5. Filter by subject and object
Get pages FROM specific domains tagged with specific terms 
<octo-query 
  what="pages" 
  by="thorped" 
  s="octothorp.es"
  o="test,demo"
  limit="5"
  autoload="true">
</octo-query>
6. Recent pages only
Filter to only show pages from the last 2 weeks 
<octo-query 
  what="pages" 
  by="posted" 
  when="recent"
  limit="10"
  autoload="true">
</octo-query>
7. Fuzzy matching
Find pages with fuzzy term matching (matches variations) 
<octo-query 
  what="pages" 
  by="thorped" 
  o="test"
  match="fuzzy-o"
  limit="5"
  autoload="true">
</octo-query>

Custom Styling

8. Dark theme
Custom CSS variables for dark mode appearance 
<style>
  octo-query.dark-theme {
    --octo-primary: #64b5f6;
    --octo-background: #1e1e1e;
    --octo-text: #e0e0e0;
    --octo-border: #333333;
  }
</style>

<octo-query 
  class="dark-theme"
  what="pages" 
  by="thorped" 
  o="demo"
  limit="3"
  autoload="true">
</octo-query>

Different Query Types

9. Get backlinks
Find pages that backlink to a specific URL 
<octo-query 
  what="pages" 
  by="backlinked" 
  o="https://octothorp.es"
  limit="5"
  autoload="true">
</octo-query>
10. Get linked pages
Find all pages linked from a specific domain 
<octo-query 
  what="pages" 
  by="linked" 
  s="octothorp.es"
  limit="5"
  autoload="true">
</octo-query>
