<octo-thorpe> Web Component

A declarative web component for displaying pages tagged with octothorpes. Built using the store pattern for minimal bundle size and maximum flexibility.

Just set HTML attributes that map directly to API parameters. No JavaScript required.

Architecture Highlights

  • Store-based: Uses Svelte stores compiled to vanilla JS (~2KB overhead)
  • Declarative: HTML attributes map 1:1 to API query parameters
  • Focused: Only queries GET /get/pages/thorped
  • Lightweight: ~15KB total (5KB gzipped) including all features
  • No dependencies: Single self-contained .js file
  • Customizable: CSS custom properties for all visual aspects

How to Use

1. Include the component

<script type="module" src="https://octothorp.es/components/octo-thorpe.js"></script>

2. Use it in your HTML

<octo-thorpe o="webdev" autoload></octo-thorpe>

3. Available Attributes

Attribute Description Default
o Octothorpe terms (comma-separated) ''
s Subject filter - domain(s) ''
noto Exclude terms ''
nots Exclude subjects ''
match Match mode (exact, fuzzy, fuzzy-o, fuzzy-s, very-fuzzy) exact
limit Maximum results 10
offset Result offset (pagination) 0
when Date filter (recent, after-DATE, before-DATE, between-DATE-and-DATE) ''
autoload Load automatically on mount false
render Display mode (list, cards, compact, count) list
server API server URL https://octothorp.es
