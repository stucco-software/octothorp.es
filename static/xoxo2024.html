######octothorp.es

Thank you for directing your browser to the kinda-secret launch of Octothorpes.

Octothorpes are hashtags and backlinks you can use on regular websites. Independent sites linked by a ring can see each other's hashtags and links, and we've opened up the first ring just for XOXO.

If you didn't get the spiel about what they are and how to use them, the demo site is a good introduction. You can also check out the octothorpe for demo.

What did I win?

Nothing! Ok, well, something. If you look under your chair, you'll find early access to the project. Just for the weekend of XOXO, any site can post to the Octothorpe network. (please be nice)

Join the world's slowest social network

If you have a website you control: You can start using Octothorpes by adding a small bit of code to the head of the page. Check out the Quickstart Guide which includes a Wordpress plugin and non-javascript options. If you want to get more technical, we've got some docs on the main site.

Honestly though, just try to find Nim or Nik and they'll set you up.

They're @nk or @nim on both the XOXO slack and xoxo.zone.

If you don't: Glitch.com can fix that! If you have a Glitch account, you can remix our starter site. It's all set up to micro- and macro-blog using Octothorpes.

Why Tweet or Toot When you can Oof!

Octothorpes doesn't store any data, but just pointing at a web site can get you a microblog's worth of content. So we built a feed for xoxo24

Oof is yet another synonym for "hashtag" so, you know, oof it up.
